Add a catalyst node

Instead of using central servers, Metacloud is run on a network of community operated nodes. These nodes store copies of all scenes deployed to Metacloud, and they handle the messaging and interactions by establishing peer-to-peer connections between users.

To propose the addition of a new node, please provide the following details.



Ethereum address of the owner of the Catalyst Node



Domain for the Catalyst Node



Description

Explain why this node should be added to the Catalyst network. Why would this addition be beneficial to the network? For example, adding nodes in new geographic areas helps to provide a better experience to more users.
